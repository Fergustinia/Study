Прокопьев Д.А. WEB-BASED PROJECT MANAGEMENT SYSTEM IMPLEMENTATION USING AGILE AND SCRUM METHODS.

Мы представляем систему управления проектами, разработанную с многоуровневой архитектурой, поддерживающую роли пользователей, планирование спринтов и отслеживание задач. Фронтенд реализован на JavaScript и HTML/CSS, в то время как бэкенд-логика и доступ к базе данных обрабатываются .NET Core REST API и PostgreSQL. Agile методология направляла разработку, а роли и церемонии Scrum были смоделированы в интерфейсе. Обратная связь пользователей подтвердила удобство использования MVP. Функциональная декомпозиция и модульный UI улучшили поддерживаемость.

Ключевые слова: Agile, Scrum, отслеживание проектов, MVP, многоуровневая архитектура.
